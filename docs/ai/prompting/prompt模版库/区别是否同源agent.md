## 如何判别是否同源

- 结构指纹：组件树是否高度一致（Header/Filters/Cards/CTA位置固定）。
- 命名指纹：CSS/类名/变量命名是否相似（如 primary/secondary, card, badge）。
- 视觉指纹：配色、圆角、阴影参数、间距是否“几乎可叠加”。
- 响应不服从：对“禁止卡片/禁止圆角/禁蓝紫”的遵从度是否一致。
- 细节缺陷：相同的拼写、相同的对齐问题或相同的空状态文案。
